## 2. 接口定义

### 2.1 接口名称: 获取资源列表
- **路径**: `GET /examples`
- **描述**: 用于查询资源的列表，支持分页过滤。

#### A. 请求参数 (Query)
| 参数名 | 类型 | 必选 | 约束 | 描述 | 示例 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| pageNumber | integer | 否 | min: 1, default: 1 | 当前页码 | 1 |
| pageSize | integer | 否 | min: 1, max: 200 | 每页条数 | 20 |

#### B. 响应报文 (Success 200)
- **Schema**: `PagedResponseDTO`
- **字段定义**:
  | 字段名 | 类型 | 必选 | 描述 | 示例 |
  | :--- | :--- | :--- | :--- | :--- |
  | data | array | 是 | 资源对象列表 | `[]` |
  | page.totalItems | integer | 是 | 总记录数 | 100 |

### 2.2 接口名称: 创建资源
- **路径**: `POST /examples`
- **描述**: 提交新资源的数据。

#### A. 请求体 (Body)
- **Schema**: `CreateRequestDTO`
- **字段定义**:
  | 字段名 | 类型 | 必选 | 约束 | 描述 | 示例 |
  | :--- | :--- | :--- | :--- | :--- | :--- |
  | name | string | 是 | maxLength: 100 | 资源名称 | "New Item" |

#### B. 响应报文 (Created 201)
- **字段定义**:
  | 字段名 | 类型 | 必选 | 描述 | 示例 |
  | :--- | :--- | :--- | :--- | :--- |
  | id | string | 是 | 系统生成的唯一标识 | "UUID-001" |
